### 🌟 Về bản thân tôi
Tôi là sinh viên chuyên ngành **Khoa học Dữ liệu (Data Science)** tại trường **Đại học Kinh tế – Đại học Đà Nẵng (DUE)**. Tôi đam mê xây dựng hệ thống dữ liệu, khai phá thông tin (Data Mining), trực quan hóa dữ liệu và phát triển các giải pháp công nghệ nhằm giải quyết các bài toán thực tế trong kinh doanh và đời sống.

Tôi luôn chủ động học hỏi, nâng cao tư duy phân tích và lập trình, hiện đang tìm kiếm các cơ hội **Thực tập (Internship)** để cống hiến và phát triển bản thân.

### 🚀 Dự án nổi bật

Dưới đây là một số dự án thực tế tôi đã xây dựng và đóng góp:

#### 1. 🏆 NextSkillAI — Nền tảng học kỹ năng số cho sinh viên
*   **Mô tả:** Đạt **Top 5 Hackathon GDGoC Viet Nam 2024**. Nền tảng cung cấp các lộ trình học kỹ năng số cá nhân hóa, tích hợp lý thuyết và trình biên dịch thực hành trực tiếp giúp sinh viên dễ dàng tiếp cận kiến thức công nghệ.
*   **Công nghệ:** Python, Django, SQL Server (Stored Procedures), HTML/CSS/JS thuần.
*   **Liên kết:** [🔍 Chi tiết mã nguồn trên GitHub](https://github.com/huybuivibecode/The-interface-for-entering-data-into-the-database-functions-like-a-POS-machine.)

#### 2. 📈 Phân tích giỏ hàng Instacart & Gợi ý sản phẩm
*   **Mô tả:** Khai phá luật kết hợp (Association Rule Mining) bằng thuật toán **Apriori** và **FP-Growth** trên 3 triệu đơn hàng của Instacart. Xây dựng Dashboard phân tích chuyên sâu trên Tableau và ứng dụng gợi ý sản phẩm tự động trên Streamlit.
*   **Công nghệ:** Python, Pandas, Mlxtend, Tableau, Streamlit.
*   **Liên kết:** [💻 Mã nguồn GitHub](https://github.com/huybuivibecode/instacart-market-basket-analysis-final) | [📊 Tableau Dashboard](https://public.tableau.com/app/profile/huy.bui6298/viz/DashboardInstacart/Dashboard1) | [🚀 Streamlit Live App](https://instacart-app-recommend-xchjf9gabmfbuyxkxaqlr4.streamlit.app/)

#### 3. 🤖 FAQ Chatbot cho Trường ĐH Kinh tế Đà Nẵng (No LLM)
*   **Mô tả:** Chatbot thông minh tự động trả lời các câu hỏi FAQ tuyển sinh và đào tạo của DUE. Sử dụng phương pháp vector hóa văn bản **TF-IDF**, lưu trữ vector trên cơ sở dữ liệu vector **ChromaDB** và tìm kiếm câu trả lời tương đồng bằng **Cosine Similarity**.
*   **Công nghệ:** Python, Streamlit, ChromaDB, Scikit-learn (TF-IDF).
*   **Liên kết:** [💻 Mã nguồn GitHub](https://github.com/huybuivibecode/Chat-box-AI-No-LLM-) | [🚀 Streamlit Live App](https://3fbhueicncrghw9cjajjzq.streamlit.app/)

#### 4. 💬 Threads Sentiment Analysis — Phân tích cảm xúc mạng xã hội
*   **Mô tả:** Hệ thống tự động thu thập dữ liệu thảo luận (comment) về các nghệ sĩ trên mạng xã hội Threads thông qua tự động hóa trình duyệt (Playwright), xử lý ngôn ngữ tự nhiên (NLP) tiếng Việt, và sử dụng **Gemini API** để phân tích cảm xúc (Tích cực / Tiêu cực / Trung lập) cùng với trực quan hóa Insights qua Plotly.
*   **Công nghệ:** Python, Playwright, Gemini API, Pandas, Plotly, Streamlit.
*   **Liên kết:** [💻 Mã nguồn GitHub](https://github.com/huybuivibecode/Sentiment-Analysis-of-Artist-Discussions-on-Threads-Using-Data-Scraping-and-Gemini-API)

#### 5. 💻 BanHang POS — Ứng dụng quản lý bán hàng
*   **Mô tả:** Hệ thống quản lý bán hàng tại quầy. Thiết kế cơ sở dữ liệu tối ưu trên SQL Server sử dụng Stored Procedures và xây dựng giao diện tương tác giống máy POS. Có tích hợp bài Lab giả lập và phòng chống tấn công SQL Injection.
*   **Công nghệ:** Python, Django, SQL Server, HTML/CSS.
*   **Liên kết:** [💻 Mã nguồn GitHub](https://github.com/huybuivibecode/The-interface-for-entering-data-into-the-database-functions-like-a-POS-machine.)
